# Eventica React - Event Discovery Platform

A modern, responsive event discovery website built with React and Vite. This is the React version featuring a beautiful light blue theme with purple gradients.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Event Slider**: Auto-scrolling carousel showcasing featured events
- **Search Functionality**: Real-time search to filter events by name, location, or category
- **Modern UI**: Clean, professional design with smooth animations
- **Dynamic Content**: Events loaded from JSON data
- **Smooth Scrolling**: Seamless navigation between sections

## Technologies Used

- **React 18**: Modern React with hooks
- **Vite**: Fast build tool and development server
- **CSS3**: Custom styling with gradients, animations, and transitions
- **JavaScript ES6+**: Modern JavaScript features

## Installation & Setup

1. Clone the repository
2. Navigate to the project directory:
   ```bash
   cd eventica-react
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## Preview Production Build

```bash
npm run preview
```
